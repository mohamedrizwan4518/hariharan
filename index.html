MY PROJECTS

Hand Gesture Recognition

Hand Gesture Recognition is a computer vision technique that detects and interprets hand movements using a camera. It involves capturing real-time video, detecting hands, identifying key landmarks, and classifying gestures to perform specific actions. Technologies like OpenCV and MediaPipe enable efficient hand tracking, while deep learning models can enhance recognition accuracy. This technology is widely used in human-computer interaction, gaming, robotics, and sign language recognition.

Platforms: Desktop and Laptop

AI Machine Learning Python Cryptography Web Development

Led Blinking Project

The LED Blinking project is a basic yet fundamental electronics project that demonstrates how to control an LED using a microcontroller like Arduino, Raspberry Pi, or ESP8266. It involves connecting an LED to a digital output pin and programming it to turn ON and OFF at regular intervals. This project helps beginners understand GPIO (General-Purpose Input/Output) control, coding logic, and timing functions. It serves as a foundation for more advanced embedded systems and IoT applications.

Platform: Hardware
